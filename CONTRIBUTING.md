# Contributing to JEDI

## Branch Strategy

- `foundation-program` — active development
- `legacy-wrapper` — frozen Astryx wrapper era

## Development

```bash
pnpm install
pnpm build
pnpm test
pnpm check:boundaries
```

## Sprint Discipline

Every PR must satisfy three pillars:

1. **Capability** — feature works
2. **Quality** — tests pass, fitness gate PASS
3. **Knowledge** — docs and Capability Matrix updated

Read [CONSTITUTION.md](./docs/CONSTITUTION.md) before contributing.
